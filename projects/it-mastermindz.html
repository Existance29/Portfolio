IT Mastermindz

An educational platform specialising in courses and resources for the IT industry

Project Overview

A website developed as a group project for the final assigment of the backend development module. Its main goal is to provide courses for various technologies and tools used in the IT industry, such as Python, AWS Cloud Computing and React.js

Tech Stack:
- HTML
- CSS
- Javascript
- NodeJS
- Express
- SQL

Courses are made and uploaded by approved users (lecturers), and are accessible to all users. Each course also has a quiz that users can participate in.
In addition, users can comment on and rate courses, encouraging engagement and feedback.
