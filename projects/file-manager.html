File Manager

A simple macOS GUI application to mass-delete files

Project Overview

This application was developed as part of my Computing+ coursework in a group project. Its a GUI tool made with python that allows users to mass delete files based on properties such as file size, creation date, etc

The goal was to provide MAC users with a user-friendly and customisable way for bulk file deletion, as most existing methods involves the command line, which is not always convenient.

Tools Used

Tkinter
Python
